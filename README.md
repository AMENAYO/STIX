# STIX Prototype

STIX is a simple front-end prototype of a Roblox-like platform.

## Features
- Create Cube or Sphere objects
- Store objects locally per user
- Login / Register
- Canvas visualization
- Theme: Yellow / Black
- Bilingual: English / Fran√ßais
- 100% Front-end: works in any browser

## How to use
1. Open `index.html` in your browser.
2. Register a new user.
3. Login with your username/password.
4. Create objects and see them in your world.
5. List your objects.
